# Example Input Brief

I want one GitHub project this week.

Constraints:

- It should come from current hot AI/open-source signals.
- It should not be a crowded generic agent platform.
- It should be small enough to ship in one week.
- It can be a skill, workflow, script, README-first product, or browser demo.
- It should have a memorable presentation and a clear market wedge.

Preferred audiences:

- AI builders
- indie hackers
- students building useful tools
- people shipping vibe-coded web apps

Output:

- 3-5 candidate ideas
- one selected build
- files to create
- README positioning
- proof that it works
